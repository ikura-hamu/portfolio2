ギャグです。

最近の LLM の発展はものすごいスピードで進んでいます。
そんな中、「AI が進化する・したから◯◯を勉強する意味は無い」というような意見を多く見かけます。
特に暗記力や計算力なんかはその標的になりやすいように思えます。
LLM に聞けばなんでも教えてもらえるから何かを暗記する意味はない、自分で計算しなくてもコンピュータが計算してくれるから計算を勉強する意味は無い、というような主張もあると思います。
この記事では、この暗記力と計算力を鍛えれば LLM に勝てるということを述べます。

## 暗記力と計算力を鍛えれば LLM に勝てる

LLM を含むニューラルネットワークの機械学習モデルは、数字の集まりであるパラメータでできており、これと入力の数字を組み合わせて計算を行い、その結果を出力しています。
LLM の出力は既知のパラメータと自分が持っている入力から計算によって得られるのですから、パラメータを全て暗記し、全て自分で計算すれば LLM と同様の結果が得られるはずです。
これを実際に短時間で実行できる人間(「大規模言語人間」と呼びます)は、今の LLM が持ってない・劣っている能力の一部(物理的な体、9.11と9.7の比較など)を持っていると考えられるため、「LLM より優れた人間」であると言えます。
従って、暗記力とか計算力を鍛えることで LLM に勝つことができるのです。

## 大規模言語人間ができること

大規模言語人間になったらどんなことができるようになるのでしょうか。
実質的に LLM が脳内に入っているのと同様なので、LLM が使えない場所で非常に役立つと考えられます。
例えばインターネットに接続できないような場所で便利な機能を使うことができます。
また、電子機器の持ち込みが禁止されている場所でも非常に役立つと考えられます。
暗記と計算でどうにかできないような試験でも、大規模言語人間は LLM に相当する計算を行うことで高得点を取れるでしょう。

## どうやったら大規模言語人間になれるのか

どれほどの暗記力と計算力を持っていれば LLM に相当する計算を自力でできるのでしょうか。

まずは暗記力についてです。OpenAI の gpt-oss-20b は、公式サイトによると 210 億個のパラメータを持っているようです。
量子化によってパラメータあたり 4.25 ビットになっているので、すべてのパラメータを 16 進数で表したとすると、およそ 236 億個の数を覚える必要があります。
https://ollama.com/library/gpt-oss:20b
似たようなものを比較対象として取り上げると、円周率の暗唱の世界記録はおよそ 10 万桁のようです。厳しそう
